# Humansort

Sometimes humans just need to sort things subjectively

## Run

Edit the file "unsorted_things.txt" with your subjective human ideas.

    python main.py

Outputs:

    [...]
    Which one is worth more (1 or 2)?
    1:  strawberry
    2:  caramel
    Answer: 1
    Which one is worth more (1 or 2)?
    1:  chocolate
    2:  strawberry
    Answer: 1
    From lowest to highest worth:
    vanilla
    caramel
    strawberry
    chocolate
